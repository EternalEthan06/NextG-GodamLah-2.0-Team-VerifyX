# VerifyX: Sovereign Digital Identity System

VerifyX is a high-assurance identity framework designed for the Malaysian landscape. It replaces vulnerable analog document processes with a secure, sovereign digital key.

## 🛡️ Multi-Layer Security Architecture
- **Layer 01 (Auth):** MFA + Biometric Liveness Detection (Face/Fingerprint).
- **Layer 02 (Deep Defense):** Cell-level AES-256 encryption.
- **Layer 03 (Accountability):** Tamper-proof WORM audit logging.
- **Layer 04 (Privacy):** "Share Capsule" technology for user-controlled consent.
